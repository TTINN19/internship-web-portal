# E-Statement Request System

A web-based system for requesting electronic bank statements (E-Statements). The platform is designed to make the statement request process more convenient, secure, and efficient by supporting document requests, downloads, email delivery, and request history tracking. :contentReference[oaicite:0]{index=0}

## Features

- Login with LDAP
- Request E-Statement by account number, date range, and language
- Download statement documents
- Send statements via email
- View request history and status
- Admin can add and unlock users :contentReference[oaicite:1]{index=1} :contentReference[oaicite:2]{index=2}

## Objectives

- Reduce manual work through automation
- Improve data security for document storage and delivery
- Support request and email history tracking
- Handle a large number of statement requests efficiently :contentReference[oaicite:3]{index=3}
