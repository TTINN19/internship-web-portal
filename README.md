# E-Statement Request System

A web-based system for requesting electronic bank statements (E-Statements). The platform is designed to make the statement request process more convenient, secure, and efficient by supporting document requests, downloads, email delivery, and request history tracking.

## Features

- Login with LDAP
- Request E-Statement by account number, date range, and language
- Download statement documents
- Send statements via email
- View request history and status
- Admin can add and unlock users

## Objectives

- Reduce manual work through automation
- Improve data security for document storage and delivery
- Support request and email history tracking
- Handle a large number of statement requests efficiently
